# Web Developer Internship Landing Page

A modern, responsive landing page for Cognifyz Technologies' Web Developer Internship program. This project showcases internship opportunities with a clean, professional design featuring animated gradients, glassmorphism effects, and mobile-first responsive layout.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean, professional design with glassmorphism effects and smooth animations
- **Animated Background**: Dynamic gradient background animation
- **Interactive Elements**: Hover effects on cards, buttons, and navigation links
- **Fast Loading**: Lightweight HTML and CSS with no external dependencies
- **SEO Friendly**: Proper semantic HTML structure and meta tags

## 🛠️ Technologies Used

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with:
  - Flexbox and Grid layouts
  - CSS Animations and transitions
  - Responsive media queries
  - Backdrop filters for glassmorphism
  - Custom properties (CSS variables)

## 🚀 How to View

1. Clone or download the project files
2. Open `index.html` in any modern web browser
3. No server setup required - works as a static website
